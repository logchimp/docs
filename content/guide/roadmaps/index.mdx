## Overview

Roadmaps represent different stages in the lifecycle of the post. Create custom roadmaps to organize, show status and allow everyone to track the progress of the posts.

Roadmaps give meaningful context to daily project work which can help motivate the team members doing it.

## Dashboard

Create and/or manage roadmaps in the **Dashboard > Roadmaps page**.

### Create roadmaps

<Alert type="tip">
	You need **`roadmap:create`** permission to perform this action.
</Alert>

Creating a new roadmap will set to private by-default. You can toggle the **Display** settings to change visibility it public.

### Update roadmaps

<Alert type="tip">
	You need **`roadmap:update`** permission to perform this action.
</Alert>

### Delete roadmaps

<Alert type="tip">
	You need **`roadmap:destroy`** permission to perform this action.
</Alert>
